\begin{DoxyReturn}{Returns}
Returns a floating-\/point number in the inclusive range {\ttfamily (0, 1)} representing the current displacement in proportion to the displacement range. If the index finger proximal joint is at the maximum displacement (rotated all the way down), {\ttfamily 1} is returned. If at the minimum displacement (all the way up, such as when the hand is held flat), {\ttfamily 0} is returned. If between the minimum and maximum, the proportion is returned. For example, half way through the displacement range (halfway between the minimum and maximum rotations/displacements), {\ttfamily 0.\+5} is returned.
