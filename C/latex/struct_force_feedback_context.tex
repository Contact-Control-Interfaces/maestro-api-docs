\subsection{Detailed Description}
A structure containing the force-\/feedback motor amplitudes for each finger. 

These are the amplitudes used to control the force-\/feedback motors for each finger. They are proportional to the power output by the motor, where {\ttfamily 255} is the maximum force and {\ttfamily 0} is no force. See {\ttfamily \hyperlink{group__force_feedback_control_ga6017d2145085cfc8f6d455a29006a640}{get\+\_\+force\+\_\+feedback\+\_\+context()}}.
